# I miei appunti di Università, fatti con Obsidian
Tutto quanto sta nella cartella Uni.
Per scaricare Obsidian visitate questo [link](https://obsidian.md/download).
Una volta aperto Obsidian cliccate "Open folder ad vault" e selezionate la cartella Uni e vi si apre tutto.
> Se vedete che potete modificare cliccate l'icona del libro in alto a destra, vicino ai 3 puntini, vi mette in read-mode
Se notate errori ditemelo che sistemo oppure modificate e fate una pull request, buono studio a tutti :)
