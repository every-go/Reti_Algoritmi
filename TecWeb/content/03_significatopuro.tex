\section{Significato di PURO (4 principi fondamentali dell'accessibilità in un sito web)}

PURO vuol dire "Percepibile, Comprensibile, Robusto, Utilizzabile".\\
Se qualcosa è PURO allora è anche accessibile, serve ad avere un metro di riferimento per capire se si è raggiunta l'accessibilità.\\

\begin{itemize}
	\item Percebile
	\begin{itemize}
		\item Fornire alternative equivalenti ad audio e video
		\item Non fare affidamento sul solo colore
		\item Usare marcatori e fogli di stile in modo approppriato
	\end{itemize}
	\item Comprensibile
	\begin{itemize}
		\item Chiarire l'uso dei linguaggi naturali
	\end{itemize}
	\item Robusto
	\begin{itemize}
		\item Trasformazione elegante, comprese le tabelle
	\end{itemize}
	\item Utilizzabile
	\begin{itemize}
		\item L'utente può tenere sotto controllo i cambiamenti dei contenuti
		\item Assicurare accessibilità diretta delle inferfacce utente incorporate
	\end{itemize}
\end{itemize}
